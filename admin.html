👑 ระบบบัญชาการ Painaidee (Admin)

🏷️ 1. ตั้งค่าหมวดหมู่ค้นหา (โชว์หน้าลูกค้า)

➕ เพิ่มหมวดหมู่ใหม่

* คำค้นหา Google Maps ต้องเป็นคำสั้นๆ กระชับ เพื่อให้แผนที่ค้นหาสถานที่เจอ

กลุ่มหลักชื่อที่แสดง (ลูกค้าเห็น)คำค้นหา (หลังบ้าน)จัดการ

📌 2. คำขอขึ้นแบนเนอร์โปรโมท (รออนุมัติ) 0

ร้านค้า (จังหวัด)หมวดหมู่ / แพ็กเกจรายละเอียดที่ขอวันที่ส่งคำขอจัดการ
ไม่มีคำขอโปรโมทในขณะนี้

✨ 3. สถานะแบนเนอร์โปรโมชั่น (กำลังแสดง) 0

* ระบบจะซ่อนแบนเนอร์ที่หมดอายุอัตโนมัติ แอดมินสามารถกดปิดซ่อน หรือแก้ไขเพิ่มวันได้

ชื่อร้าน (หมวดหมู่)รายละเอียดแบนเนอร์เวลาหมดอายุสถานะจัดการ
ไม่มีแบนเนอร์ที่กำลังแสดงผล

📝 4. คำขอลงทะเบียนร้านค้าใหม่ 0

ชื่อร้านรูปภาพแนบจัดการ
ไม่มีคำขอลงทะเบียนใหม่

🏬 5. ฐานข้อมูลร้านค้าทั้งหมด (จัดการ PIN / VIP)

ชื่อร้านรหัส PINสถานะพิเศษจัดการ
ยังไม่มีร้านค้าในระบบ

🚩 6. รายงานร้านปิดกิจการถาวร 0

* หากกด "ยืนยันว่าปิด" ระบบจะซ่อนร้านนี้ไม่ให้แสดงในแผนที่แอปอีกต่อไป

ชื่อร้านที่ถูกแจ้งวันที่ลูกค้าแจ้งมาจัดการ (แบนออกจากระบบ)
ไม่มีรายงานร้านปิดถาวร
